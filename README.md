# Job Portal Application

A simple PHP-based job portal application for connecting job seekers and employers.

## Features

- **Job Seekers**:
    - Create and manage your profile.
    - Upload CV/Resume.
    - Search for jobs by title or location.
    - Apply for jobs and track application status.
- **Employers**:
    - Build company profiles.
    - Post and manage job openings.
    - View and manage job applications.
- **Analytics**:
    - Market insights and job distribution data.

## Installation

1. Clone the repository.
2. Ensure you have a PHP environment with MySQL.
3. Import the `db_job_seeker.sql` into your database.
4. Configure database settings in `config/env.php`.
5. Start the application using a local server or Docker.

### Docker Support

You can run the application using Docker Compose:

```bash
docker-compose up -d
```

Access the application at `http://localhost:8004` (or the configured port).
